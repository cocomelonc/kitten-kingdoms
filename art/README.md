# Cover artwork

`kitten-kingdoms-cover.png` is original project key art generated with the
built-in OpenAI image-generation tool on 2026-07-22. The only visual inputs
were this repository's previous cover, current Android gameplay screenshot,
and original kitten sprite sheet. No stock image or third-party character art
was used. The generated 16:9 result was resized to 1280x720 without changing
its composition.
